Features / Components

1.User Accounts
Client

  * Browse products
  * Add products to favorites
  * Ask about products via WhatsApp
  * Measure ring size through a dedicated page
  * Search via Categories or Product's name

Admin

  * Perform all client tasks
  * Create new product categories
  * Add or delete products



 2. Backend

* Built with Laravel
* Handles authentication, product management, and categories
* Link : https://github.com/cuervo0102/GoldBijoux



3. Frontend

* Built with Android Studio (Java)
* Uses Retrofit for connecting to the backend
* To connect your local device or emulator to the backend, update the **IP address** in `RetrofitClient.java` inside the `network` folder



4. Database

* MySQL: is used to store users, products, categories, and other data
